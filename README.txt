Automaticlly adds a flexible Back to Top button to your WordPress website that allows your visitor to scroll back to the top of your page with one click!


== Installation ==

**Through Dashboard**

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **ws-scroll-top** in the search box and click on search button.
3. Find ID ws-scroll-top plugin.
4. Then click on Install Now after that activate the plugin.
5. Go to Settings -> Scroll Down to Show Right Corner.

**Installing Via FTP**

1. Download the plugin to your hardisk.
2. Unzip.
3. Upload the "ws-scroll-top" folder into your plugins directory.
4. Log in to your WordPress admin panel and click the Plugins menu.
5. Then activate the plugin.
6. Go to Website -> Scroll Down to Show Right Corner.
